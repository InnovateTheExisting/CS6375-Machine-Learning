Instructions to execute this project:
1) The program can be executed within python3.6 environment.
2) Open Command Prompt(Windows)/Terminal(Mac OS) and navigate to the current location of this project folder.
3) Type name of the file and six parameters to run this program.

The sample of test command is,
python yxs173130-hw1.py <L> <K> <training-set> <validation_set> <test-set> <to-print>

Because the two data sets are along with .py file, so we can use relative path to load data.
The example test commands what I used are given below,
python yxs173130-hw1.py 40 10 ./data_sets1/training_set.csv ./data_sets1/validation_set.csv ./data_sets1/test_set.csv no
python yxs173130-hw1.py 30 20 ./data_sets1/training_set.csv ./data_sets1/validation_set.csv ./data_sets1/test_set.csv yes
python yxs173130-hw1.py 28 12 ./data_sets2/training_set.csv ./data_sets2/validation_set.csv ./data_sets2/test_set.csv no
python yxs173130-hw1.py 30 10 ./data_sets2/training_set.csv ./data_sets2/validation_set.csv ./data_sets2/test_set.csv yes
