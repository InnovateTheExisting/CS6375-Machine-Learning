Instructions to execute this project:
1) The program can be executed within python3.6 environment.
2) Open Command Prompt(Windows)/Terminal(Mac OS) and navigate to the current location of this project folder.
3) Type name of the file (or parameters) to run this program.

The sample of test command for Naive Bayes is,
python NB.py <if will remove stop words>
The example test commands what I used are given below,
python NB.py no
python NB.py yes

The sample of test command for Logistic regression is,
python LR.py <lambda> <if will remove stop words>
The example test commands what I used are given below,
python LR.py 0.1 no
python LR.py 0.1 yes
